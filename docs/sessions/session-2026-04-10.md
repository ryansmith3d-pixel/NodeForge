## Architecture Established

**Pipeline model:** Color Array → Array Assign → Write → JSON. Schema drives cardinality.
Color Array "Match Schema" button auto-sizes rows to match Schema field count. Array Assign
warns visually on mismatch.

**Port type vocabulary (locked):**

| Type | Description | Produced by |
|---|---|---|
| `color` | Single hex value | Color Swatch |
| `color_array` | Ordered list of (label, hex) pairs | Color Array |
| `token_role` | Single role name string | Schema (future per-role ports) |
| `token_dict` | Full token object | Schema (current, Gang only) |
| `assignment` | (role, hex) pair | Assign or Array Assign |

**Wire system:** Cubic bezier curves. Typed port enforcement at drag time — incompatible
targets render muted/red. Port dots filled when connected, hollow when empty. Wire color
from token file (`edge.default` / `edge.selected`).

**Write node:** Traverses actual wired edges only. No scene scanning. Graph is the
source of truth.

**Node view switching:** Every node has a button strip. View state is per-instance.
Node height and port positions recalculate on every view switch.

**Spawn behavior:** New nodes appear at viewport center. Successive nodes offset +20px X,
-20px Y. Cascade resets on manual drag.

**QColorDialog fix:** `parent=None` + `DontUseNativeDialog` flag required on Windows to
prevent blank picker dialog.
